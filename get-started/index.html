Get started

Follow these steps to create your antwar site.

npm install antwar-cli -g
antwar --init <folder name>
cd <folder name>
antwar --develop

Then add pages to the pages folder and blog posts to the posts folder. When you're ready, run antwar --build to generate the static files.

Posts

Posts are added as .md files to the /posts directory. You need to put a frontmatter block in the beginning of each file. We have put a couple of example posts in there that you can look at for inspiration.

Drafts

If you need a place for your posts before they are finished, you should put them in the /drafts folder. They will show up in the development environment so you can see exactly how they will look when you publish them.

Pages

Pages are React components or .md files. Put them in the /pages folder!
